## 工厂模式 Factory

> 工厂模式可以替代类实例化时的 new 方法；


```
class A
{
    //不允许类不直接实例化 或克隆
    private function __construct(){}
    private function __clone(){}
}

class B
{
    private function __construct(){}
    private function __clone(){}
}


class Factory
{
    public static function getInstance($class)
    {
        //类对象的实例化通过工厂类产生
        return new $class();
    }
}

$a = Factory::getInstance('A');
$b = Factory::getInstance('B');
```
